Introduction

What is Clip Path?

Based on Mozilla,The clip-path CSS property creates a clipping region that sets what part of an element should be shown. Parts that are inside the region are shown, while those outside are hidden.

What is Clip Helper?

Well the clip path sometimes quite hard to be created. So simply Clip Helper is a free CSS libarary that has 40 classes (for now) to help you with some clip path. I got some of the shape from other's (Don't forget to check the credits)

Version?

The version of this library is 1.0

How to start?

After downloading you are free to edit the css file or maybe if you are in a hurry, just include it and find the class you need in the reference or type the class you are looking for in the search box at the bottom of your screen and then hit the enter key

Example

To give an element a clip-path just use the class name example:

<div class="blackBox diagonal-1"></div>

That line of code will give this output

Sometimes you will need to give a shape outside for the element. so when you give it a float, it will become like this

Then you will need to add the class name with '-shaped' to the end of the class name. I'll give you an example

<div class="blackBox diagonal-1 diagonal-1-shaped"></div>
<div class="blackBox diamond diamond-shaped"></div>
<div class="blackBox home home-shaped"></div>

Warning!!!

Becareful for some classes they would behave differently depends on the width and height for example the blackBox class is 100x100 px. I recommend not to use non-rectangular class for elements that do not have the same width and height
